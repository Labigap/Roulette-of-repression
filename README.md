# Roulette of repression

## Описание 
Это приложение, в котором Вы - власть страны и Вы должны решать судьбу приложений, сервисов, людей и организаций.

Приложение находится в разработке, на данный момент работает только РКН Рулетка и рулетка арестов, но в скором времени будут открыты все режимы!

## Запуск проекта
1. Заходите в папку bin ➡️ debug
2. Находите и скачиваете RouletteOfRepression.exe
3. Запускаете его
4. Наслаждайтесь игрой :)
