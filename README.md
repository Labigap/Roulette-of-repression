# Roulette of repression

### Описание 
Это приложение, в котором Вы - власть страны и Вы должны решать судьбу приложений, сервисов, людей и организаций.

Статус: готово к использованию

На данный момент открыты все режимы, игра ожидает рефакторинг и мелкие изменения

### Запуск проекта
1. Заходите в папку bin ➡️ debug
2. Находите и скачиваете RouletteOfRepression.exe
3. Запускаете его
4. Наслаждайтесь игрой :)
5. Если вы открываете программу по тегу v1.0, то для начала нужно зайти в папку "Source code.zip"

### Системные требования:
##### Рекомендуемые требования:
1. Разрешение экрана: 1366х768 пикселей
2. Операционная система: Windows 7
3. Процессор: Любой, от 1ггц и выше
4. Оперативная память: 2gb
5. Место на диске: 500мб (если не установлен .NET Framework) 
6. Дополнительно: .NET Framework 4.7.2 и выше
