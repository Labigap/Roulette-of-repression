# Roulette of repression

### Описание 
Это приложение, в котором Вы - власть страны и Вы должны решать судьбу приложений, сервисов, людей и организаций.

Статус: В разработке...

Приложение находится в разработке, на данный момент работает только РКН Рулетка, Рулетка арестов и Преследование НКО, 
но в скором времени будут открыты все режимы!

3. Запускаете его
4. Наслаждайтесь игрой :)
